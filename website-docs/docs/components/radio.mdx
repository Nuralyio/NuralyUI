# Radio

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/radio";
import { HyRadio } from "@nuralyui/radio/react";
import { useState } from "react";

The Radio component allows users to select one option from a set of mutually exclusive options. It's perfect for forms where only one choice should be made from multiple alternatives.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/radio
```

</TabItem>
 <TabItem value="yarn" label="yarn">

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/radio';
</script>

<hy-radio-input 
  options='[{"label": "Option 1", "value": "opt1"}, {"label": "Option 2", "value": "opt2"}]'
  defaultValue="opt1">
</hy-radio-input>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HyRadio } from '@nuralyui/radio/react';

const options = [
  { label: 'Option 1', value: 'opt1' },
  { label: 'Option 2', value: 'opt2' },
  { label: 'Option 3', value: 'opt3' }
];

function App() {
  return (
    <HyRadio 
      options={options}
      defaultValue="opt1"
      onChange={(value) => console.log('Selected:', value)}
    />
  );
}
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `options` | `RadioButtonOption[]` | `[]` | Array of radio options |
| `type` | `'default' \| 'button'` | `'default'` | Style type of the radio buttons |
| `position` | `'left' \| 'right'` | `'left'` | Position of the radio button relative to label |
| `direction` | `'vertical' \| 'horizontal'` | `'vertical'` | Layout direction of the radio group |
| `defaultValue` | `string` | `undefined` | Default selected value |

### Types

#### RadioButtonOption

```typescript
interface RadioButtonOption {
  label: string;
  value: string;
  disabled?: boolean;
}
```

### Events

| Event | Description | Event Detail |
|-------|-------------|--------------|
| `change` | Fired when selection changes | `{ value: string }` |

### CSS Custom Properties

You can customize the radio appearance using CSS custom properties:

```css
hy-radio-input {
  --radio-size: 16px;
  --radio-border: 1px solid #d9d9d9;
  --radio-border-checked: #1890ff;
  --radio-background-checked: #1890ff;
  --radio-dot-color: #ffffff;
  --radio-label-margin: 8px;
  --radio-disabled-color: #00000040;
  --radio-disabled-background: #f5f5f5;
  
  /* Button type specific */
  --radio-button-background: #ffffff;
  --radio-button-border: 1px solid #d9d9d9;
  --radio-button-checked-background: #1890ff;
  --radio-button-checked-color: #ffffff;
  --radio-button-hover-border: #40a9ff;
}
```
