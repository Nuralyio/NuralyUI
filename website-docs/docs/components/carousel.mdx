# Carousel

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/carousel";
import { HyCarousel } from "@nuralyui/carousel/react";
import { useState } from "react";

The Carousel component allows you to display a series of content in a rotating slideshow format. Perfect for showcasing images, testimonials, product features, or any content that benefits from a sequential presentation.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/carousel
```

</TabItem>
 <TabItem value="yarn" label="yarn">

```shell
yarn add @nuralyui/carousel
```

</TabItem>
</Tabs>

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/carousel';
</script>

<hy-carousel autoplay="true" interval="3000">
  <div slot="slide">
    <img src="image1.jpg" alt="Slide 1" />
  </div>
  <div slot="slide">
    <img src="image2.jpg" alt="Slide 2" />
  </div>
  <div slot="slide">
    <img src="image3.jpg" alt="Slide 3" />
  </div>
</hy-carousel>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HyCarousel } from '@nuralyui/carousel/react';

const slides = [
  { id: 1, content: <img src="image1.jpg" alt="Slide 1" /> },
  { id: 2, content: <img src="image2.jpg" alt="Slide 2" /> },
  { id: 3, content: <img src="image3.jpg" alt="Slide 3" /> }
];

function App() {
  return (
    <HyCarousel 
      slides={slides}
      autoplay={true}
      interval={3000}
      showDots={true}
      showArrows={true}
    />
  );
}
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `slides` | `CarouselSlide[]` | `[]` | Array of slides to display |
| `autoplay` | `boolean` | `false` | Enable automatic slide progression |
| `interval` | `number` | `3000` | Auto-play interval in milliseconds |
| `showDots` | `boolean` | `true` | Show navigation dots |
| `showArrows` | `boolean` | `true` | Show navigation arrows |
| `infinite` | `boolean` | `true` | Enable infinite loop |
| `pauseOnHover` | `boolean` | `true` | Pause autoplay on hover |
| `direction` | `'horizontal' \| 'vertical'` | `'horizontal'` | Slide direction |
| `effect` | `'slide' \| 'fade'` | `'slide'` | Transition effect |
| `height` | `string` | `'auto'` | Carousel height |
| `speed` | `number` | `500` | Transition speed in milliseconds |
| `touchSwipe` | `boolean` | `true` | Enable touch/swipe navigation |

### Types

#### CarouselSlide

```typescript
interface CarouselSlide {
  id: string | number;
  content: React.ReactNode | string;
  alt?: string;
  caption?: string;
}
```

### Events

| Event | Description | Event Detail |
|-------|-------------|--------------|
| `slide-change` | Fired when active slide changes | `{ currentSlide: number, previousSlide: number }` |
| `autoplay-start` | Fired when autoplay starts | `{}` |
| `autoplay-stop` | Fired when autoplay stops | `{}` |

### Methods

| Method | Parameters | Description |
|--------|------------|-------------|
| `goToSlide(index)` | `index: number` | Navigate to specific slide |
| `nextSlide()` | None | Go to next slide |
| `previousSlide()` | None | Go to previous slide |
| `startAutoplay()` | None | Start autoplay |
| `stopAutoplay()` | None | Stop autoplay |

### CSS Custom Properties

You can customize the carousel appearance using CSS custom properties:

```css
hy-carousel {
  --carousel-background: #ffffff;
  --carousel-border-radius: 6px;
  --carousel-arrow-color: #ffffff;
  --carousel-arrow-background: rgba(0, 0, 0, 0.5);
  --carousel-arrow-hover-background: rgba(0, 0, 0, 0.7);
  --carousel-dot-color: rgba(255, 255, 255, 0.5);
  --carousel-dot-active-color: #ffffff;
  --carousel-dot-size: 12px;
  --carousel-transition-duration: 0.5s;
  --carousel-transition-easing: ease-in-out;
}
```
