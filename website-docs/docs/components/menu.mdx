# Menu

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/menu";
import { HyMenu } from "@nuralyui/menu/react";

The Menu component provides a flexible navigation menu system. Perfect for application navigation, context menus, and dropdown menus with hierarchical structure.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/menu
```

</TabItem>
 <TabItem value="yarn" label="yarn">

```shell
yarn add @nuralyui/menu
```

</TabItem>
</Tabs>

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/menu';
</script>

<hy-menu orientation="vertical">
  <hy-menu-item>Home</hy-menu-item>
  <hy-menu-item>About</hy-menu-item>
  <hy-menu-item>Contact</hy-menu-item>
</hy-menu>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HyMenu } from '@nuralyui/menu/react';

function App() {
  return (
    <HyMenu orientation="vertical">
      <hy-menu-item>Home</hy-menu-item>
      <hy-menu-item>About</hy-menu-item>
      <hy-menu-item>Contact</hy-menu-item>
    </HyMenu>
  );
}
```

</TabItem>
</Tabs>

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Menu orientation |
| `disabled` | `boolean` | `false` | Whether the menu is disabled |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Size of the menu |

### Events

| Event | Description |
|-------|-------------|
| `select` | Fired when a menu item is selected |
| `change` | Fired when selection changes |

## Examples

### Horizontal Menu

```html
<hy-menu orientation="horizontal">
  <hy-menu-item>Home</hy-menu-item>
  <hy-menu-item>Products</hy-menu-item>
  <hy-menu-item>Services</hy-menu-item>
  <hy-menu-item>Contact</hy-menu-item>
</hy-menu>
```

### Vertical Menu

```html
<hy-menu orientation="vertical">
  <hy-menu-item>Dashboard</hy-menu-item>
  <hy-menu-item>Users</hy-menu-item>
  <hy-menu-item>Settings</hy-menu-item>
  <hy-menu-item>Logout</hy-menu-item>
</hy-menu>
```

### Menu with Icons

```html
<hy-menu orientation="vertical">
  <hy-menu-item icon="home">Home</hy-menu-item>
  <hy-menu-item icon="user">Profile</hy-menu-item>
  <hy-menu-item icon="settings">Settings</hy-menu-item>
</hy-menu>
```
