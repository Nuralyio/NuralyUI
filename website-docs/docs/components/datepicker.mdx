# DatePicker

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/datepicker";
import { HyDatePicker } from "@nuralyui/datepicker/react";

The DatePicker component allows users to select dates from a calendar interface. It supports single date selection, date ranges, different formats, and localization.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/datepicker
```

</TabItem>
 <TabItem value="yarn" label="yarn">

```shell
yarn add @nuralyui/datepicker
```

</TabItem>
</Tabs>

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/datepicker';
</script>

<hy-datepicker 
  value="2023-12-01"
  format="YYYY-MM-DD">
</hy-datepicker>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HyDatePicker } from '@nuralyui/datepicker/react';

function App() {
  return (
    <HyDatePicker 
      value="2023-12-01"
      format="YYYY-MM-DD"
    />
  );
}
```

</TabItem>
</Tabs>

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `value` | `string` | `undefined` | The selected date value |
| `format` | `string` | `'YYYY-MM-DD'` | Date format pattern |
| `placeholder` | `string` | `'Select date'` | Placeholder text |
| `disabled` | `boolean` | `false` | Whether the datepicker is disabled |
| `readonly` | `boolean` | `false` | Whether the datepicker is read-only |
| `min` | `string` | `undefined` | Minimum selectable date |
| `max` | `string` | `undefined` | Maximum selectable date |

### Events

| Event | Description |
|-------|-------------|
| `change` | Fired when the date selection changes |
| `input` | Fired during date input |

## Examples

### Basic DatePicker

```html
<hy-datepicker placeholder="Select a date"></hy-datepicker>
```

### With Default Value

```html
<hy-datepicker value="2023-12-25"></hy-datepicker>
```

### Date Range Restrictions

```html
<hy-datepicker 
  min="2023-01-01" 
  max="2023-12-31"
  placeholder="Select date within 2023">
</hy-datepicker>
```

### Custom Format

```html
<hy-datepicker 
  format="DD/MM/YYYY"
  placeholder="DD/MM/YYYY">
</hy-datepicker>
```

### Disabled State

```html
<hy-datepicker 
  disabled 
  value="2023-12-01"
  placeholder="This is disabled">
</hy-datepicker>
```
