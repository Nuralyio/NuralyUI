# Select

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/select";
import { HySelect } from "@nuralyui/select/react";
import { useState } from "react";

The Select component allows users to choose one or multiple options from a dropdown list. It supports various configurations including single/multi-select, different sizes, and custom styling.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/select
```

</TabItem>
 <TabItem value="yarn" label="yarn">

```shell
yarn add @nuralyui/select
```

</TabItem>
</Tabs>

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/select';
</script>

<hy-select 
  placeholder="Select an option"
  options='[{"label": "Option 1", "value": "opt1"}, {"label": "Option 2", "value": "opt2"}]'>
</hy-select>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HySelect } from '@nuralyui/select/react';

const options = [
  { label: 'Option 1', value: 'opt1' },
  { label: 'Option 2', value: 'opt2' },
  { label: 'Option 3', value: 'opt3' }
];

function App() {
  return (
    <HySelect 
      placeholder="Select an option"
      options={options}
      onSelectionChange={(selectedValues) => console.log(selectedValues)}
    />
  );
}
```

</TabItem>
</Tabs>

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `options` | `IOption[]` | `[]` | Array of options to display |
| `placeholder` | `string` | `'Select an option'` | Placeholder text when no option is selected |
| `disabled` | `boolean` | `false` | Whether the select is disabled |
| `selectionMode` | `'single' \| 'multiple'` | `'single'` | Selection mode for the select |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Size of the select component |
| `status` | `'default' \| 'error' \| 'warning'` | `'default'` | Status variant of the select |
| `type` | `'default' \| 'borderless'` | `'default'` | Style type of the select |
| `defaultSelected` | `string[]` | `[]` | Array of default selected option values |
| `show` | `boolean` | `false` | Controls whether the dropdown is open |

### Types

#### IOption

```typescript
interface IOption {
  label: string;
  value: string;
  disabled?: boolean;
}
```

### Events

| Event | Description | Event Detail |
|-------|-------------|--------------|
| `selectionChange` | Fired when selection changes | `{ selectedValues: string[], selectedOptions: IOption[] }` |

### CSS Custom Properties

You can customize the select appearance using CSS custom properties:

```css
hy-select {
  --select-background: #ffffff;
  --select-border: 1px solid #d9d9d9;
  --select-border-radius: 6px;
  --select-border-hover: #40a9ff;
  --select-border-focus: #1890ff;
  --select-padding: 4px 11px;
  --select-font-size: 14px;
  --select-placeholder-color: #bfbfbf;
  --select-option-hover-background: #f5f5f5;
  --select-option-selected-background: #e6f7ff;
  --select-dropdown-shadow: 0 6px 16px 0 rgba(0, 0, 0, 0.08);
}
```
