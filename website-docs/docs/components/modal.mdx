# Modal

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";
import "@nuralyui/modal";
import { HyModal } from "@nuralyui/modal/react";

The Modal component provides a way to display content in a dialog overlay, capturing user attention and focus. It's perfect for confirmations, forms, or any content that needs to interrupt the user's workflow.

## How to use it

<Tabs>
  <TabItem value="npm" label="npm">

```shell
npm install @nuralyui/modal
```

</TabItem>
 <TabItem value="yarn" label="yarn">

```shell
yarn add @nuralyui/modal
```

</TabItem>
</Tabs>

## Import

<Tabs>
  <TabItem value="html" label="HTML">

```html
<script type="module">
  import '@nuralyui/modal';
</script>

<hy-modal open="false">
  <div slot="content">
    <h2>Modal Title</h2>
    <p>This is the modal content.</p>
  </div>
</hy-modal>
```

</TabItem>
  <TabItem value="react" label="React">

```jsx
import { HyModal } from '@nuralyui/modal/react';

function App() {
  return (
    <HyModal open={false}>
      <div slot="content">
        <h2>Modal Title</h2>
        <p>This is the modal content.</p>
      </div>
    </HyModal>
  );
}
```

</TabItem>
</Tabs>

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `open` | `boolean` | `false` | Whether the modal is visible |
| `closable` | `boolean` | `true` | Whether the modal can be closed by clicking outside or pressing escape |
| `title` | `string` | `undefined` | The title of the modal |

### Slots

| Slot | Description |
|------|-------------|
| `content` | The main content area of the modal |

### Events

| Event | Description |
|-------|-------------|
| `close` | Fired when the modal is closed |
| `open` | Fired when the modal is opened |

## Examples

### Basic Modal

```html
<hy-modal open="true" title="Confirmation">
  <div slot="content">
    <p>Are you sure you want to proceed?</p>
    <button>Yes</button>
    <button>No</button>
  </div>
</hy-modal>
```

### Form Modal

```html
<hy-modal open="true" title="Edit Profile">
  <div slot="content">
    <form>
      <label>Name: <input type="text" /></label>
      <label>Email: <input type="email" /></label>
      <button type="submit">Save</button>
    </form>
  </div>
</hy-modal>
```
